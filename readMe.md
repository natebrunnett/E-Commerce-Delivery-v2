E-Commerce-Delivery-v2 w/ Stripe and AdminBro (localhost only)
	
 	AdminBro
 		$cd admin-bro-localhost
   		$npm install
     		Read README.md
       		$nodemon

	Server
		$cd server
		$npm install
		$nodemon

	Client
		$cd client
		$npm install
		$npm start

	Important:
		*Requires MongoDB to be installed and running on OS
